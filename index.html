개인정보 처리방침

시행일: 2026년 3월 26일

DK SYSTEM(이하 "회사")은 「간편견적서」(이하 "앱") 이용자의 개인정보를 소중히 보호합니다. 본 개인정보 처리방침은 앱이 어떤 정보를 수집하고 어떻게 처리하는지 안내합니다.

핵심 요약: 간편견적서는 모든 데이터를 사용자의 기기에만 저장하며, 외부 서버로 어떤 데이터도 전송하지 않습니다. 분석 도구, 광고 SDK, 제3자 서비스를 사용하지 않습니다.

1. 수집하는 정보

앱은 사용자가 직접 입력하는 다음 정보를 기기 내에 저장합니다:

  • 회사 정보: 상호, 사업자등록번호, 대표자명, 주소, 연락처, 이메일, 업태, 업종
  • 거래처 정보: 상호, 사업자등록번호, 대표자명, 연락처, 이메일, 주소, 메모
  • 견적서 정보: 견적서 제목, 번호, 품목 내역, 금액, 메모, 상태
  • 이미지: 회사 로고 및 직인 이미지 (사용자가 직접 선택하거나 앱 내 생성)

2. 정보 저장 방식

모든 데이터는 Apple SwiftData 프레임워크를 통해 사용자의 기기 내부에만 저장됩니다.

  • 외부 서버로 데이터를 전송하지 않습니다.
  • 회사가 운영하는 별도의 서버는 없습니다.
  • iCloud 동기화를 활성화한 경우, Apple의 iCloud 서비스를 통해 사용자의 다른 기기와 동기화될 수 있습니다. 이는 Apple의 개인정보 처리방침에 따릅니다.

3. 사진 라이브러리 접근

앱은 회사 로고 및 직인 이미지를 등록하기 위해 사진 라이브러리 접근 권한을 요청할 수 있습니다.

  • 사용자가 직접 선택한 이미지만 접근합니다.
  • 선택한 이미지는 기기 내에만 저장되며, 외부로 전송되지 않습니다.
  • 사진 라이브러리의 다른 이미지에는 접근하지 않습니다.

4. 제3자 제공 및 공유

앱은 어떤 개인정보도 제3자에게 제공하거나 공유하지 않습니다.

  • 광고 SDK를 사용하지 않습니다.
  • 분석(Analytics) 도구를 사용하지 않습니다.
  • 사용자 행동 추적을 하지 않습니다.

5. PDF 공유

사용자가 PDF 공유 기능을 사용할 경우, iOS 시스템의 공유 시트(Share Sheet)를 통해 카카오톡, 이메일 등 사용자가 선택한 앱으로 PDF 파일이 전달됩니다. 이는 사용자의 명시적 행위에 의해서만 실행되며, 자동으로 전송되지 않습니다.

6. 알림(Notification)

견적서 유효기간 알림 기능은 iOS 로컬 알림을 사용합니다. 알림은 기기 내에서 처리되며, 외부 푸시 서버를 사용하지 않습니다.

7. 데이터 삭제

사용자는 언제든지 앱의 설정 화면에서 모든 데이터를 삭제할 수 있습니다. 앱을 삭제하면 기기에 저장된 모든 데이터가 함께 삭제됩니다.

8. 아동의 개인정보

앱은 만 14세 미만 아동의 개인정보를 수집하지 않습니다. 앱의 특성상 사업자를 대상으로 하며, 아동이 사용할 것으로 예상하지 않습니다.

9. 암호화

앱은 자체적으로 암호화 기능을 사용하지 않습니다(ITSAppUsesNonExemptEncryption = NO). 데이터 보호는 iOS 기기 자체의 암호화 기능에 의존합니다.

10. 개인정보 처리방침 변경

본 개인정보 처리방침이 변경될 경우, 앱 업데이트를 통해 안내드립니다. 변경 사항은 앱 내 설정 화면에서 확인할 수 있습니다.
